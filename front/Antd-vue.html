  • 入门配置

# 入门配置

  • 官方网址 https://3x.antdv.com/docs/vue/introduce-cn
  • 使用 npm 或 yarn 安装
  • 以下内容基于Ant Design Vue3.2.20版本所编写
npm install ant-design-vue --save
yarn add ant-design-vue

# 注册并使用

  • 如果使用 Vue 默认的模板语法，需要注册组件后方可使用
  • 全局完整注册
    • 样式文件需要单独引入
//在main.js中配置
import Vue from 'vue';
import Antd from 'ant-design-vue';import App from './App';
import 'ant-design-vue/dist/antd.css';

const app = createApp(App);

app.use(Antd).mount('#app');

  • 组件内部注册

    • 此种方式需要分别注册组件子组件，并且注册后仅在当前组件中有效
    <template>
  <a-button>Add</a-button>
</template>
<script>
  import { Button } from 'ant-design-vue';
  const ButtonGroup = Button.Group;

  export default {
    components: {
      AButton: Button,
      AButtonGroup: ButtonGroup,
    },
  };
</script>

# 国际化配置

  • 官方文档：https://3x.antdv.com/docs/vue/i18n-cn
  • ant-design-vue 提供了一个 Vue 组件 ConfigProvider 用于全局配置国际化文案。 ConfigProvider 不包含时间类组件的国际化，你需要额外引入时间库(dayjs、momentjs、date-fns 等)的国际化文件。
<template>
  <a-config-provider :locale="locale === 'en' ? enUS : zhCN">
    <App />
    {/* 以下切换语言功能在组件里配置 */}
    <div class="change-locale">
    <span style="margin-right: 16px">Change locale of components:</span>
      <a-radio-group v-model:value="locale">
        <a-radio-button key="en" :value="enUS.locale">English</a-radio-button>
        <a-radio-button key="cn" :value="zhCN.locale">中文</a-radio-button>
      </a-radio-group>
    </div>
  </a-config-provider>
</template>

<script>
  import enUS from 'ant-design-vue/es/locale/en_US';
  import zhCN from 'ant-design-vue/es/locale/zh_CN';
  import dayjs from 'dayjs';
  import 'dayjs/locale/zh-cn';
  import { defineComponent, ref, watch } from 'vue';

  dayjs.locale('en');

  export default defineComponent({
    setup() {
      const visible = ref(false);
      const locale = ref(enUS.locale);
      watch(locale, val => {
        dayjs.locale(val);
      });
      return {
        locale,
        dayjs,
        enUS,
        zhCN,
      }
    }
  })
</script>
